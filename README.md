# Anna-project

Простая и красивая игра в крестики-нолики на Python с графическим интерфейсом.

## Запуск

Убедитесь, что у вас установлен Python 3 с `tkinter`, затем выполните:

```bash
python3 tic_tac_toe.py
```

На macOS также можно запустить файл `launch_tic_tac_toe.command` двойным кликом или командой:

```bash
./launch_tic_tac_toe.command
```

Также доступен `.app`-лаунчер: `Tic Tac Toe.app`.

## Возможности

- современный неоновый интерфейс;
- подсветка победной комбинации;
- счётчик побед и ничьих;
- кнопки для новой партии и полного сброса счёта.
